01. 성남시청 Seongnam City Hall

제작기간 : 3주 / 적응형

Using : HTML, CSS, JavaScript, jQuery

웹 사이트를 제작하는 것이 익숙하지 않아 구현하는데 고생했지만, 콘텐츠를 하나씩 구현할 때마다 재미와 성취감을 느끼게 해준 작업이었습니다.

첫 프로젝트였기에 기존 웹 사이트의 있는 모든 기능을 구현하여 자기 것으로 만들기 위해 플러그인을 사용하지 않고 구현했습니다.
기억에 남는 기능으로 CSS animation을 사용해 구현한 검색창의 움직이는 그라데이션 테두리, jQuery를 사용해 구현한 클릭 시 아래에서 위로 올라오는 하단 메뉴 등이 있습니다.

웹 접근성을 위해 skip 메뉴와 IE9 버전까지 크로스 브라우징을 통해 구현했고 각 메인 메뉴마다 1개씩 서브 페이지를 제작했습니다.

02. 멜론 Melon

제작기간 : 2주 / 반응형

Using : HTML, CSS, JavaScript, jQuery

웹 접근성을 고려하고 코드를 작성한 프로젝트입니다.
skip 메뉴 구현, title 속성, 이미지의 alt 속성을 작성하고 IR 기법을 통해 이미지의 대체 텍스트 삽입, IE9 버전까지 크로스 브라우징으로 구현했습니다.

두 번째 작업이었던 만큼 원활하게 진행됐고 빠른 작업을 위해 slick, marquee 플러그인을 사용하여 슬라이드, 흐르는 텍스트를 구현했습니다.
기억에 남는 기능으로 jQuery를 사용해 구현한 실시간 검색어 차트와 Media Queries를 사용한 반응형 웹 구축 등이 있습니다.

기존 웹 사이트와 다르게 PC 레이아웃을 기반으로 반응형 웹을 구현했습니다.
